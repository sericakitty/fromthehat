This is a probability calculator.
You have change to choose different kind of ball and number of balls and put them into the hat.
Next you will choose type of ball you want to draw and number of balls you want to draw.
Program will calculate the probability of drawing the ball you expected.


This section defines a type of balls and how many of them you put inside a hat. First choose a type of ball you want to add to the hat.
Next input the number of balls you want to add to the hat. If you choose the same type of ball again, the number of balls will be updated.
Lastly click the "Insert inside Hat list" button to add the ball to the hat.

This section defines the balls you are expecting to get when you are pulling balls from the hat.
First choose from the list ball you want to get. If you choose the same ball again, the number of balls will be updated.
Next give number how many of them you are expecting to get.
Lastly click the "Insert inside expected list" button to add the ball to the expected balls.
